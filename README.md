# Practica Spring

Proyecto de ejemplo creado con Spring Boot para demostrar `Controller`, `RestController` y plantillas Thymeleaf.

- Enlace de GitHub del usuario: https://github.com/CrisGood/Practica-spring

Instrucciones rápidas:

1. Compilar y ejecutar:

```bash
mvn spring-boot:run
```

2. Endpoints disponibles:

- Página de inicio: http://localhost:9090/ 
- Información: http://localhost:9090/informacion
- Saludo: http://localhost:9090/saludo
- Gestión de usuarios: http://localhost:9090/gestionusuarios
- API saludo: http://localhost:9090/api/saludo
- API usuarios: http://localhost:9090/api/usuarios
- API greeting: http://localhost:9090/api/greeting
- API time: http://localhost:9090/api/time

3. Captura de pantalla:

- Añada su captura en `screenshots/screenshot.png` y haga commit.
